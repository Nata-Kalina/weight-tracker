About

Hi! Welcome to the Weight Tracker App!

If you want to get healthier and lose weigth, you have to stick to this process. You have to make a goal and track the progress towards that goal. The good approach is to visualize the progress that helps you to stay motivated and stay on track and continue to push those numbers.

I tried to do my best and create Weight Tracker app using Node.JS, Express Framework and MongoDB with Mongoose library as a data storage. Front end was build with HTML, CSS and JavaScript and it uses API with fetch() calls. Back end has 3 Mongoose data models: Users, Weight Goals and Current Weight.

Other packages that were used:

  • Security packages:
    • Helmet
    • Cors
    • Xss-clean
    • Express-rate-limit
  • Dotenv NPM package
  • Jsonwebtoken package for user authentication
  • Bcryptjs/dist/bcrypt for storing user's passwords hashed and salted
